Add a real Comments Moderator Role to your powerful WordPress Blog!

== Description ==

Just install the plugin, activate it: You can now change a user's role to "Moderator", he can now moderate all comments, and only do this ;)

== Installation ==

1. Upload the *"baw-moderator-role"* folder into the *"/wp-content/plugins/"* directory
1. Activate the plugin through the *"Plugins"* menu in WordPress
1. You can now change a user's role to "Moderator", he can now moderate all comments, and only do this ;)
